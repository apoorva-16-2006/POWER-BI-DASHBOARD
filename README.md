# POWER-BI-DASHBOARD
# 📊 Data Analytics Dashboard Collection

This repository contains a collection of **interactive Power BI dashboards** built using different datasets to analyze business performance, sales trends, and customer behavior.

The purpose of this project is to demonstrate **data analytics, visualization, and data storytelling skills** using Power BI.

---

## 🚀 Dashboards Included

### 1️⃣ Sales Superstore Dashboard
Analyzes overall **sales performance across regions, categories, and states** to identify trends and business insights.

#### Key Insights
- Total Sales and Average Order Value  
- Sales by Region  
- Sales by Category and Subcategory  
- Sales distribution across states  
- Delivery time analysis  
- Sales by Ship Mode  

#### Visualizations Used
- KPI Cards  
- Bar Charts  
- Line Charts  
- Maps  
- Tables  
- Category Filters  

---

### 2️⃣ Amazon Sales & Reviews Analytics Dashboard
Analyzes **product performance, pricing, discounts, ratings, and reviews** from an Amazon dataset.

#### Key Insights
- Total Revenue generated  
- Number of products and users  
- Average discount percentage  
- Product rating distribution  
- Price distribution of products  
- Category and subcategory performance  
- Relationship between discount and product rating  

#### Visualizations Used
- KPI Cards  
- Scatter Plot  
- Histogram  
- Area Charts  
- Bar Charts  
- Category Analysis Charts  

---
